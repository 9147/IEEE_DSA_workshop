# IEEE CS DSA WORKSHOP

This repository contains the materials and resources from the Data Structures and Algorithms (DSA) workshop conducted by the IEEE Computer Society.

## About the Workshop

The DSA workshop is an initiative by the IEEE Computer Society to help students enhance their knowledge and skills in Data Structures and Algorithms. The workshop covers a wide range of topics, from basic data structures like arrays, linked lists, stacks, queues, trees, and graphs, to  algorithms and problem-solving strategies.

## Contents

The repository includes:

- Lecture notes
- Code samples
- Practice problems
- Solutions to practice problems
- Additional resources for further learning

## Usage

To make the most of this repository, start by reviewing the lecture notes to understand the concepts.
